# Actual Solution Skeleton

This package contains the downloadable starter solution.

## What is included

- solution file
- four main projects
- three test projects
- starter controllers
- tenant middleware
- local authentication command flow
- JWT service
- project command/query example
- repository and Dapper read model starter
- explicit unit of work
- starter domain model

## What you still need to implement

- EF Core database scaffolding from your SQL Server
- refresh token storage and rotation
- Entra token validation and exchange endpoint
- MFA challenge flow
- permission loading from database
- audit log persistence
- field-level authorization filters
- request/response body logging with masking
- threat model and hardening review

## How to use

1. Download the zip.
2. Extract it.
3. Open the `.sln` in Visual Studio 202x.
4. Restore packages.
5. Update `appsettings.json`.
6. Scaffold database objects into Infrastructure.
7. Start implementing the TODO sections.
